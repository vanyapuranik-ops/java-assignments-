# Assignment 1: Menu-Driven Calculator in Java

## Objective

To implement a menu-driven calculator that performs basic arithmetic operations.

## Concepts Used

* Classes and Objects
* Methods
* Switch Case
* Loops
* Scanner

## Description

This program allows users to perform addition, subtraction, multiplication, division, and modulus operations using a menu-driven approach.

The program runs continuously until the user exits.

## How to Run

1. Compile: javac Calculator.java
2. Run: java Calculator
